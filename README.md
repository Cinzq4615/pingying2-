# 小学语文看拼音写词语生成器 (AI Pinyin Worksheet Generator)

这是一个基于 **Google Gemini AI** 的智能化教育辅助工具。它可以自动识别课本生词表图片，生成标准的带声调拼音，并排版成 A4 规格的田字格练习卷，支持直接导出 PDF 打印。

## 📂 项目文件说明 (核心配置文件)

为了让项目在本地正常运行，我们使用了以下三个关键配置文件：

1.  **`package.json`**:
    *   **作用**：项目的“户口本”。定义了项目名称、版本以及它依赖的“零件”（如 React, Gemini SDK）。
    *   **用法**：通过它，你可以使用 `npm install` 一键安装所有依赖。
2.  **`tsconfig.json`**:
    *   **作用**：TypeScript 的“规则书”。由于我们使用了更严谨的 TypeScript 语言，这个文件规定了代码如何被翻译成浏览器能懂的 JavaScript。
3.  **`vite.config.ts`**:
    *   **作用**：项目的“超级管家”。它负责启动极速开发服务器，并把你的 API Key 安全地注入到代码中。

## 🚀 本地运行指南

### 第一步：环境准备
安装 [Node.js](https://nodejs.org/)（建议选择 LTS 版本，v18.0 以上）。

### 第二步：下载与安装
1. 将项目文件夹下载到本地。
2. 打开终端（或 CMD），进入项目根目录。
3. 执行安装命令：
   ```bash
   npm install
   ```

### 第三步：配置 API KEY (至关重要 ⚠️)
1. 访问 [Google AI Studio](https://aistudio.google.com/) 免费获取你的 API Key。
2. 在项目根目录下创建一个名为 `.env` 的新文件（注意前面有个点）。
3. 在 `.env` 文件中写入以下内容：
   ```env
   VITE_API_KEY=将这里替换为你的实际API_KEY
   ```
   *注意：不要在 Key 前后加空格或引号。*

### 第四步：启动项目
在终端输入：
```bash
npm run dev
```
终端会显示一个地址（通常是 `http://localhost:5173`），按下 `Ctrl` 并点击链接，或者在浏览器中手动输入该地址即可访问。

## 📖 使用流程
1. **上传图片**：点击“添加图片”按钮，上传课本后的生词表（支持多张）。请确保图片清晰、水平。
2. **生成练习卷**：点击“开始生成”，AI 会自动解析词语并标注拼音。
3. **预览与修改**：在页面下方查看生成的 A4 预览效果。
4. **导出 PDF**：点击“导出高质量 PDF”，保存到本地。

---

## ⚠️ 特别注意点 (必读)

### 1. 🖨️ 打印设置 (最重要)
在打印生成的 PDF 时，请务必在打印对话框中进行如下设置：
*   **缩放比例 (Scale)**: 选择 **“实际大小” (Actual Size)** 或 **“100%”**。
*   **切勿选择“适合页面”**: 否则田字格的尺寸会缩小，导致练习效果不佳。
*   **纸张选择**: 确保设置为 **A4**。

### 2. 🤖 AI 识别提示
*   **儿化音**：系统已强制 AI 将“哪儿”拆分为“nǎ er”两个格，以符合小学生书写习惯。
*   **图片质量**：如果 AI 识别出错，通常是因为图片太模糊或光线不均匀，建议重新拍照上传。

### 3. 🛡️ API 安全
*   `.env` 文件包含敏感的 API Key。**请勿**将其上传到 GitHub 等公开平台。
